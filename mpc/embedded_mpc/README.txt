OSQP Embedded Solver for MPC
========================================

Generated by: generate_embedded_solver.py
Date: 2025-10-10

This is a malloc-free, library-free embedded C solver.

To compile:
  cd embedded_mpc
  make

To integrate with Rust firmware:
  1. Copy files to src/firmware/mpc_solver/
  2. Create FFI bindings in build.rs
  3. Call from mpc.rs module
